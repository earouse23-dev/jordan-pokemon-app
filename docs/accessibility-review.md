# Accessibility review

Implemented: landmarks and headings; skip link; semantic buttons/inputs; persistent form labels; visible focus; keyboard-openable ledger rows; meaningful image alternatives; aria-live result/toast regions; dialog semantics; Escape/backdrop close; focus restoration; minimum practical action heights; no color-only price status; reduced-motion support; no hover-only or drag-only operations; camera has file/manual alternatives.

Manual verification still required on iOS VoiceOver, Android TalkBack, Windows screen reader, 200% zoom, and physical keyboard. Production charts need visible timeframe/units and a summary table when real history exists. Focus trapping should be upgraded when the sheet grows beyond this product slice.
