# Product strategy

## Value proposition

“From physical card to trustworthy collection record—with the market context and provenance needed to make an informed decision.”

## Position

Mica is an independent collector ledger, not a marketplace, game, appraisal service, grading predictor, or universal price oracle.

## Revenue assumptions

Entitlements prepare for monthly scan count, collection size, refresh frequency, saved views, history range, import size, and alerts. Pricing is deliberately undecided. Existing user data remains accessible during provider outages or plan changes.

## Risks

Provider licensing and continuity; exact-printing recognition; false confidence around price/condition; card-image rights; paid AI abuse; mobile capture privacy; and user trust when data is stale or partial.

## Retention loop

Acquire card → scan/confirm → record ownership → see attributable collection context → research/maintain → return for next acquisition or refresh.
