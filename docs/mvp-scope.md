# MVP scope

## Launch

Account creation/login/reset, private capture, staged recognition, human confirmation, manual search, add/edit owned copies, mobile ledger, desktop table adaptation, search/filter/sort/saved views, card detail, attributable current quotes, partial totals, safe CSV import/export, offline/recoverable states, usage controls, monitoring, RLS, deletion/export, and accessibility verification.

## Implemented product slice

All primary UI states and local interactions above except connected auth, remote OCR/vision, production provider calls, server imports, and scheduled snapshots. Those infrastructure-dependent capabilities have schema/contracts/documented acceptance criteria.

## Deferred

Transaction-level history, graded-price provider, alerts, insurance reports, public sharing, households, native shells, condition estimation, grading prediction, sealed products, other games, dealer commerce, and billing activation.

## Excluded without explicit approval

Scraping, automated trade execution, “real-time universal” claims, casual-photo condition claims, appraisals, manipulated community pricing, and copied proprietary history.
